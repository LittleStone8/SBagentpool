# Python 屎上最不走心的代理池 项目

------
## ———— 善待每一个代理



### 直接谈一下项目吧，不聊技术（反正也没人看代码）：

#### 首先先说代理的存储方式吧，储存在redis中。

所有代理的信息都只存放在一个有序集合（SortedSet），一个哈希表(Hash)，一个链表（List）中。<br>
有序集合：存放分数低于100分的代理，按分数排序，分数越低越优质。<br>
哈希表：存放代理的详细信息。<br>
链表：存放分数高于100分（确认死得透透）的代理。<br>
